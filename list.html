Notes on list in HTML

Ordered list

Ordered list are used to display a list of items in a specific order.The 'ol' tag is used to create an ordered list.Each item is enclosed in'li' tag

  1. Item 1
  2. Item 2
  3. Item 3

Unordered list

Unordered list are used to display a list of items in no specific order.This'ul' tag is used to create an unordered list.Eaach item in the list is enclosed in an 'li' tag

  • Item 1
  • Item 2
  • item 3
